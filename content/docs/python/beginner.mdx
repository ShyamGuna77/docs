Welcome! This is a short, hands-on crash course to get you comfortable writing your first Python programs.

## Install Python

- Windows/macOS/Linux: download from [python.org](https://www.python.org/downloads/).
- After installing, verify:
```bash
python --version
# or sometimes:
python3 --version
```

## Your first script

Create a file named `hello.py` and add:
```python
print("Hello, world!")
```

Run it:
```bash
python hello.py
# or:
python3 hello.py
```

## Use the REPL (interactive shell)

Open the Python prompt:
```bash
python
```
Then try:
```python
2 + 2
print("hi")
```
Exit with `exit()` or `Ctrl + Z` then Enter on Windows (`Ctrl + D` on macOS/Linux).

## Variables and basic types

```python
age = 21              # int
pi = 3.14159          # float
name = "Ada"          # str
is_admin = False      # bool

print(type(age))      # <class 'int'>
print(type(name))     # <class 'str'>
```

### Strings
```python
greeting = f"Hello, {name}!"
print(greeting.lower())
print("abc".upper())
print("ha" * 3)       # "hahaha"
```

## Input and output

```python
user = input("What's your name? ")
print("Nice to meet you,", user)
```

## Collections: lists and dictionaries

```python
# List
nums = [3, 1, 4]
nums.append(1)
nums.sort()
print(nums)  # [1, 1, 3, 4]

# Dictionary (key-value pairs)
person = {"name": "Ada", "age": 36}
print(person["name"])
person["job"] = "Engineer"
print(person)
```

## Control flow

```python
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
else:
    grade = "C"

print(grade)
```

## Loops

```python
# for
for n in [1, 2, 3]:
    print(n)

# while
count = 3
while count > 0:
    print(count)
    count -= 1
print("Go!")
```

## Virtual environments (recommended)

Keep project dependencies isolated.

```bash
# create
python -m venv .venv

# activate
# Windows (PowerShell):
.venv\Scripts\Activate.ps1
# Windows (CMD):
.venv\Scripts\activate.bat
# macOS/Linux:
source .venv/bin/activate

# install packages
pip install requests

# freeze dependencies
pip freeze > requirements.txt
```

Deactivate anytime with:
```bash
deactivate
```

## Mini-project: Number guessing game

Create `guess.py`:




```python
import random

print("I'm thinking of a number between 1 and 20.")
secret = random.randint(1, 20)
attempts = 0

while True:
    guess_text = input("Take a guess: ")
    if not guess_text.isdigit():
        print("Please enter a number.")
        continue

    guess = int(guess_text)
    attempts += 1

    if guess < secret:
        print("Too low!")
    elif guess > secret:
        print("Too high!")
    else:
        print(f"You got it in {attempts} tries! The number was {secret}.")
        break
```

Run it:
```bash
python guess.py
```
